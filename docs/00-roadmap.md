# Portfolio v2 — Roadmap

## Objetivo

Construir un portafolio de nivel Top 1%, con estándares de ingeniería comparables a los utilizados en empresas como Vercel, Stripe y Linear.

---

## Principios

- Calidad sobre velocidad.
- Código limpio y mantenible.
- Accesibilidad primero.
- Performance como prioridad.
- Componentes reutilizables.
- Arquitectura escalable.
- Consistencia visual.
- Animaciones con propósito.
- SEO técnico completo.
- Mobile First.
