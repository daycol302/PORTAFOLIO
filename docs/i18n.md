# Internacionalización

El portafolio ofrece español (`es`) e inglés (`en`), con español como idioma predeterminado. Las cadenas de interfaz se concentran en `src/i18n/dictionary.ts` y se exponen mediante `I18nProvider` y `useI18n`.

La preferencia se guarda durante un año en la cookie `portfolio-locale` y en `localStorage`. En la primera visita se usa el idioma del navegador cuando está soportado. El selector actualiza el atributo `lang` del documento sin cambiar la ruta, por lo que se conserva el ancla y la navegación actual.

## Añadir un idioma

1. Añada el código a `locales` en `src/i18n/config.ts`.
2. Cree una entrada con la misma estructura que `es` en `src/i18n/dictionary.ts`.
3. Añada la opción correspondiente al selector.
4. Revise metadata, textos de datos de cada feature y la accesibilidad.

Para un texto nuevo, declare la clave en ambos diccionarios y consúmala con `const { dictionary } = useI18n()`. No incluya cadenas visibles directamente en componentes.
